# ClinicaAPI
Se tiene la necesidad de realizar una aplicación web, que permita registrar las citas médicas de una  clínica, donde los pacientes reciben su consulta médica mediante la programación de una cita. En  dicha cita se vincula al paciente y al doctor o especialista para una fecha y hora. Después de la cita  el doctor registra el diagnóstico del paciente.
